
# Paradise Nursery

Paradise Nursery is a React and Redux shopping application for browsing houseplants and managing a shopping cart.

Last Updated: 7/18/2026

## Project Features

- Landing page with a background image, company name, company description, and Get Started button.
- Product listing page with plants grouped into categories.
- Eighteen unique houseplants total, with six plants in each of three categories.
- Each product displays a thumbnail, name, and price.
- Add to Cart buttons add items to the Redux cart and become disabled after the item is added.
- Header appears on both the product listing page and cart page.
- Header includes links to Home, Plants, and Cart.
- Cart icon dynamically displays the total number of items in the cart.
- Shopping cart page displays:
  - Total number of plants.
  - Total cart cost.
  - Thumbnail, name, unit price, quantity, and item total for each plant.
  - Increase, decrease, and delete buttons.
  - Continue Shopping button.
  - Checkout button that displays a Coming Soon message.

## Technologies Used

- React
- Redux Toolkit
- React Redux
- React Router
- Vite
- CSS

## Run Locally

```bash
npm run dev
```

Then open the local URL shown in the terminal.
